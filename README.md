# Funny Paper Jumping Jacks

Static website project for an English-language presentation of antique Wissembourg paper jumping jacks.

## Structure
- `index.html` – homepage
- `collection.html` – overview of categories
- `about.html` – collector background
- `categories/` – one page per category
- `assets/` – styles, scripts and images

## Publish
This repository is intended for GitHub Pages.
